# Live AI Dashboard

A visually stunning, responsive dashboard that monitors a simulated Voice AI agent session.

## Project Overview

This project is a frontend challenge that demonstrates complex UI development, real-time state simulation, and effective use of AI coding agents.

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **UI Library**: React 19
- **Styling**: Tailwind CSS with Dark Mode
- **Language**: TypeScript
- **Code Quality**: ESLint, Prettier

## Getting Started

### Prerequisites

- Node.js 18.17.0 or later
- npm or yarn package manager

### Installation

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
- `npm run type-check` - Run TypeScript type checking
