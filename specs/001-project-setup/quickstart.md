# Quickstart: Project Setup & Infrastructure

**Feature**: 001-project-setup
**Created**: 2025-12-31

## Setup Instructions

### Prerequisites

- Node.js version 18.17.0 or later installed
- npm or yarn package manager installed
- Git for version control

### Initialize Project

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:3000
```

### Verify Installation

1. Development server starts without errors
2. Dark-themed page loads in browser
3. Project structure includes:
   - `components/` directory
   - `hooks/` directory
   - `lib/` directory

### Code Quality Tools

```bash
# Format code
npm run format

# Check types
npm run type-check

# Lint code
npm run lint
```

## Verification Checklist

- [ ] Development server starts successfully
- [ ] Default page renders with dark mode
- [ ] All required directories exist
- [ ] Code formatting tools work
- [ ] Type checking runs without errors
- [ ] Linting provides clear feedback
